# Event Watchlist

The event watchlist is a list of records consisting of a name and optionally email address and date of birth, that you as the event organiser can add to which will be actively monitored by Solidrock.

Any ticket purchased that has a match with a record on the watchlist will be flagged in the system for your attention. This is useful for example if you have a list of known troublemakers or banned individuals that you want to be alerted about if they attempt to purchase a ticket for your event. You can then choose to cancel the order or take any other appropriate action.

Records added to the watchlist for an event will automatically be included in future event watchlist's unless they are explicitly removed. This means you can build up a watchlist over time and it will be applied to all your events without you having to add the same records in for each event. You can of course remove any records from the watchlist at any time if you no longer want them to be monitored for your events.
